# SheBuilds

SheBuilds is a web-based Kudumbasree Management System developed using Django to digitize and streamline administrative and community-level operations.

## Features

- Admin, User, Authority, CDS, ADS, and Kudumbasree modules
- Role-based authentication and authorization
- Complaint and feedback management
- Service and product booking
- ADS approval for Kudumbasree registrations
- Notification management
- Ward and locality-based administration
- Document upload and verification

## Technology Stack

- Python
- Django
- SQLite
- HTML
- CSS
- JavaScript
- Bootstrap

## Modules

- Admin Module
- User Module
- Authority Module
- Kudumbasree Module
- CDS Module
- ADS Module

## Objective

To provide a centralized platform for managing Kudumbasree activities, improving transparency, accountability, and communication among all stakeholders.
